服务路径：
/etc/systemd/system/screen-monitor.service

服务脚本：
[Unit]
Description=Monitor keyboard activity to manage screen state
After=multi-user.target

[Service]
ExecStart=/home/admin/autolcd.sh
Restart=always

[Install]
WantedBy=multi-user.target


开机自动启动
sudo systemctl enable screen-monitor.service
运行脚本
sudo systemctl start screen-monitor.service
停止脚本
sudo systemctl stop screen-monitor.service

重启脚本
sudo systemctl restart screen-monitor.service
